# Safeguard - YouTube Parental Control

Safeguard is a comprehensive parental control application designed to provide a secure and monitored environment for children to watch YouTube videos.

## Features

- **Parent Dashboard**: Monitor watch history, set time limits, and manage allowed channels/categories.
- **Child Interface**: A simplified, kid-friendly video player with restricted content access.
- **YouTube Integration**: leverages the YouTube Data API to fetch and filter safe content.
- **Analytics**: Usage statistics and viewing habits.

## Tech Stack

- **Backend**: Node.js, Express, TypeScript
- **Frontend**: React, Vite, TypeScript, Tailwind CSS

## Setup Instructions

### Backend
1. Navigate to `backend/`
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure your keys.
4. Run `npm run dev`

### Frontend
1. Navigate to `frontend/`
2. Run `npm install`
3. Copy `.env.example` to `.env` and configure.
4. Run `npm run dev`
