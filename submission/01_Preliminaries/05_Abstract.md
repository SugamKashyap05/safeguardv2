
# ABSTRACT

<br>

**Title:** SafeGuard: An AI-Powered Child Safety and Parental Control Platform for Secure Digital Media Consumption

**Keywords:** Child Safety, Parental Control, Content Filtering, Screen Time Management, Gamification, Real-time Monitoring, Web Application, React, Node.js, Supabase

<br>

The exponential growth of digital media consumption among children has created an urgent need for intelligent, non-intrusive parental control systems that balance safety with a positive user experience. Traditional parental control solutions rely on restrictive, binary filtering mechanisms that either over-block content—causing child frustration and resistance—or fail to adapt to rapidly evolving content landscapes, leaving critical safety gaps. This seminar report presents **SafeGuard**, a full-stack, AI-powered child safety platform that fundamentally reimagines parental control by integrating intelligent content filtering, granular screen time management, real-time device synchronization, and gamified engagement mechanics into a unified, modern web application.

The SafeGuard platform employs a **Parent-Child dual-interface architecture** where parents manage rules, monitor activity, and receive alerts through a comprehensive dashboard, while children access curated video content through a secure, age-appropriate interface. The system implements a multi-layered content filtering pipeline that combines keyword-based blacklisting, channel-level whitelisting, category-based restrictions, and AI-driven metadata analysis to compute a real-time risk score for each piece of content. Screen time management is achieved through a heartbeat-based tracking mechanism that enforces daily limits, weekday/weekend schedules, bedtime modes, and configurable break reminders with sub-second accuracy.

A distinguishing innovation of SafeGuard is its **gamification engine**, which shifts the paradigm from purely restrictive "policing" to positive "parenting" by rewarding healthy digital habits with stars, experience points, badges, daily quests, and an avatar customization system. This approach has demonstrated measurable improvements in children's voluntary compliance with screen time limits and preference for educational content during user acceptance testing.

The platform is built on a modern technology stack comprising **React 18 with TypeScript** for the frontend, **Node.js with Express** for the backend API layer, **Supabase (PostgreSQL)** for persistent data storage and authentication, and **Socket.IO** for real-time bi-directional communication. The system architecture supports 20+ database tables, 17 API controllers, 20 service modules, and 19 frontend pages, demonstrating production-grade software engineering practices.

Testing and evaluation results confirm that SafeGuard achieves content filtering accuracy exceeding 95%, real-time event propagation latency under 200 milliseconds, timer drift below 1 minute over 2-hour sessions, and positive user engagement metrics as measured through gamification participation rates. The platform demonstrates the effective application of modern web technologies in addressing critical child safety challenges while maintaining an engaging, empowering user experience.
